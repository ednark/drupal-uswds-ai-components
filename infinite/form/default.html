USWDS Form — Drupal Custom Twig + Form API

Requires overriding multiple form-related Twig templates to map Drupal's Form API to USWDS classes.

Form builder (PHP)

$form['name'] = [
  '#type' => 'textfield',
  '#title' => $this->t('Full Name'),
  '#attributes' => ['class' => ['usa-input']],
  '#label_attributes' => ['class' => ['usa-label']],
];

Template override

{# input.html.twig #}
<input{{ attributes.addClass('usa-input') }} />
{# label attributes are applied automatically #}
