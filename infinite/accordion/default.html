USWDS Accordion — Drupal Paragraph Bundle

Provided by uswds_paragraph_components_accordion submodule.

Installation

drush pm:enable uswds_paragraph_components_accordion

Usage

  1. Add a Paragraphs field to your content type
  2. Select USWDS Accordion as an allowed bundle
  3. Editors add accordion items as child paragraphs
  4. Each item has a heading and body text

Template

Override: paragraph--uswds-accordion.html.twig

Output: <ul class="usa-accordion"> <li class="usa-accordion__item"> ...
