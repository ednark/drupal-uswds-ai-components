Data Visualizations — Drupal Field Type (plotly_js)

Provided by the plotly_js module maintained by smustgrave.

Installation

composer require drupal/plotly_js
drush pm:enable plotly_js
drush plotly_js:download-plotly
drush cache:rebuild
